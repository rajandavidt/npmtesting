# cnw

> Made with create-react-library

[![NPM](https://img.shields.io/npm/v/cnw.svg)](https://www.npmjs.com/package/cnw) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save cnw
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'cnw'
import 'cnw/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
